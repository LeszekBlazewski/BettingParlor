BettingParlor_Refactor Namespace

Contains all classes and forms which are necessary for the application to work properly.
Classes
  ClassDescription
Bet
Contains definition for all kinds of bets in project.
BetDummy
Extension of the Bet class. Fake bet used when player does not place bet or all bets are reset after the race.
BetFactory
Factory class. Returns a new instance of Bet derivatives.
BetHandicap
Extension of the Bet class. Special type of bet used when player places handicap bet.
BetStandard
Extension of the Bet class. Special type of bet used when player places standard bet.
Bettor
Extension of the User class. Contains all methods for performing basic bettor operations.
Client
Class contains methods which are used to send requests to server and receive the response. Class handels all client operations which relate to server-client communication. All methods are implemented by using SignalR framework.
ClientServerConnectionsHub
Inherits from Hub class implemented in SignalR framework. Connection hub which is the base for all server-client methods used in the programm. Contains methods which are used to properly manage all events related to client requests and responses. Class contains all server methods which are used by administrator to push information to clients.
DataClassesBettingParlorDataContext
Main context which allows access to all data stored in database.
Dog
Provides base fields and methods which are used to properly handle dog pictureBoxes on the form.
FormAddCash
Represents a dialogbox which is used to get the amount of cash which player wants to tranfer to his account.
FormClient
Represents a window which is the base graphic interface for each client who managed to connect to server. Calls methods by the use of controllers after client interacts with the form.
FormLogin
Represents a window which allows each user to sign in into his account or create new one. All loggin and registration functions are called from this form.
FormMenu
Menu where user decides what other forms he wants to see. Contains button which direct to specific forms.
FormPlaceBet
Shown to player only when he wants to place a bet. Used to get dog number and value of the bet which should be placed.
FormRules
Contains only one label which describes the rules of the game.
FormServer
Represents a window which is the base graphic interface for administrator who handles all the operations.
GreyHound
Extension of Dog class. Contains all methods used to move pictureBoxes on the form.
LoadUserData
Extension of the UserData class. Contains all methods responsible for loading user data from SQL dataBase to application.
Login
Contains methods used to handle the registration and loggin process.
ManageUserData
Contains methods which process user data stored in database.
Parlor
Main administrator class. Contains all methods used by administrator to manage the race and take part in the game.
Program
Base class created by .NET framework.
Server
Contains all methods which are used to host and close the server properly.
Startup
Class is neccessary for SignalR framework to work properly.
User
Base class for are players in the application. Contains a skeleton of methods for derived classes to implement them correctly.
UserData
Stores all user data contained in static dictionary which is used by derived classes to access player information whenever needed.
